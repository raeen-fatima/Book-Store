# Book-Store

A simple online book store application built using [MERN Stack].

## Features:
- Browse books by categories.
- User authentication for login and registration.
- Book purchase functionality.

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/raeen-fatima/Book-Store.git
   
2. Navigate to the directory:
   cd Book-Store

4. Install dependencies:
  npm install

5. Run the application:
   npm start

[React/Node.js/Express/etc.]
